# StartWithGroovySpringElasticSearch
Dummy Groovy REST App with SpringBoot / SpringData / ElasticSearch

## Pre-requisites
 - Java 7
 - Gradle 2.1

## Step-by-step
 - $ git clone <this repo>
 - $ cd <this repo>
 - $ gradle build
 - $ gradle run
 - Enjoy on http://localhost:8080/
